What is Fiasco-Mustache ?

"Fiasco-Mustache" is a project I made in order to easily produce playsets for Fiasco, the game by Bully Pulpit Game.

This project is a web-page, with no server-related code, so everyone can get a local copy and test it with any web-browser. The name comes from a distinct library used to generate the playset : Mustache.

How to use Fiasco-Mustache ?

Download the zip file, unzip it, and double-click on generator.html.

or test it online

Then follow the instructions in the "About" tab.
